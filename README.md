# Credit Card Customer Churn Analysis

## Project Overview

This project analyzes customer churn behavior in a credit card business using R and Power BI. The objective is to identify factors influencing customer attrition and develop a predictive model to help improve customer retention.

## Business Problem

Customer churn results in revenue loss and increased customer acquisition costs. Understanding the factors that contribute to customer attrition can help businesses develop targeted retention strategies.

## Tools and Technologies

* R
* R Markdown
* ggplot2
* dplyr
* Logistic Regression
* Power BI

## Dataset

The dataset contains customer demographic information, account details, and transaction behavior data for over 10,000 credit card customers.

Key variables include:

* Attrition_Flag
* Customer_Age
* Total_Trans_Amt
* Total_Trans_Ct
* Months_Inactive_12_mon
* Gender
* Income_Category
* Card_Category

## Project Workflow

1. Data Loading and Cleaning
2. Exploratory Data Analysis (EDA)
3. Logistic Regression Model Development
4. Model Evaluation using Confusion Matrix
5. Power BI Dashboard Development
6. Business Recommendations

## Key Findings

* Customer inactivity is strongly associated with churn.
* Customers with lower transaction counts are more likely to leave.
* Transaction behavior is an important predictor of customer retention.
* Approximately 17% of customers in the dataset had churned.

## Model Performance

* Algorithm: Logistic Regression
* Approximate Accuracy: 83%
